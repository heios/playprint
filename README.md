# playprint

A single **static HTML page** (publishable to GitHub Pages, no server) that generates playful
children's **cut-out card templates** — month names, digits, or any text — and exports them as a
**multi-page PDF**.

Type or pick a set of words; every whitespace-separated token becomes a bordered, playfully-tilted
card with a friendly handwriting font. Add a coloured **mat** behind each card, choose a paper size
(A0–A10, B0–B10, US Letter/Legal/Tabloid), and download a print-ready PDF. Work is saved as named
projects in the browser and can be shared by URL.

The full product spec lives in **[SPEC.md](SPEC.md)** — the single source of truth.
